1. Hackathon Paradigm Alignment & Core Purpose

The SWB Remote Pivot Engine v3.0 is engineered explicitly to align with the core objective of the Scale Without Borders AI Hackathon: identifying high-quality, legitimate remote employment paths in tech while shielding immigrant talents from malicious market vulnerabilities. By combining strict identity verification with advanced, deterministic audit checking, the platform prevents fraud at the entry layer and accelerates technical integration.

2. Cryptographic Role-Based Access Control (RBAC) Specification

The system enforces an absolute system-level access boundary. Loose, shifting frontend client switches are entirely deprecated in favor of explicit state containers. Users must verify their profiles via simulated authentication handshakes before accessing specialized workspace actions.

2.1 State-Transition Model

The system operates on three strict enumerated configuration spaces:

  • `UNASSIGNED`: The initial application gateway view container. Completely wraps and hides underlying processing pipelines. Restricts interaction to verified identity validation cards.
  • `ROLE_JOB_SEEKER`: Authorized upon simulated Google or LinkedIn federated authentication. Unlocks resume optimization arrays, industry keyword targeting engines, and download loops for copyable code delivery blueprints.
  • `ROLE_JOB_POSTER`: Authorized upon validating an enterprise corporate email address. Grants permission to execute corporate infrastructure scoring routines, startup runway evaluation audits, and remote-readiness verification checks.

3. Recruiter/Poster Integrity & Validation Framework

When an authenticated entity with `ROLE_JOB_POSTER` attempts to submit an opportunity, the payload passes through a strict deterministic and heuristic grading matrix to generate an aggregate Legitimacy Score (0–100%).

3.1 The Legitimacy Logic Matrix

Verification Directive Scoring Allocation Automated Compliance Rule Mechanics
Domain Integrity Handshake +20 Score Points Automated alignment evaluation: Extract email suffix and verify it matches the root domain of the corporate website.
Careers Portal Alignment +25 Score Points Validates that the specific opening is mirrored natively on the company’s official internal index paths, ensuring legitimacy.

Financial Extortion Override Forfeit to 0% Pattern matching safety switch: Triggers immediate isolation if text requests hardware deposits, registration fees, or training costs.

3.2 Growth-Tier Classifications

Organizational scale properties are evaluated strictly via numeric parameters to prevent data contradictions:

  1. Startup: 1 to 99 employees. Enforces specialized evaluation of venture funding variables (`fundingInfo`).
  2. Mid-Market Enterprise: 100 to 999 employees. Evaluated against verified mid-tier operational footprints.
  3. Large Enterprise / MNC: 1,000+ employees. Assessed against global operational multi-national delivery standards.

3.3 Rigid Borderless Remote-Only Enforcement Policy

To preserve alignment with the hackathon's core premise, the submission process blocks hybrid or localized configurations. If the incoming payload parameters describe an on-site or hybrid setup, the interface intercepts the call, blocks API network dispatching, and displays a strict theme-compliance validation error.
